1 Затянуть все последние изменения
git fetch -p

2 Переключаемся на последнюю копию удаленного мастера
git checkout origin/master

3 создаем новую ветку
git checkout -b issue name

4 сделать задание

5 Проверить статус
git status

6 Все созданные или измененные файлы красненькие добавить в гит
git add имя файла

7 закоммитить изменения
git commit -a -m "человеку понятное описание"

8 Сделать pull request 
git push origin имя ветки

9 взять ссылку из консоли и открыть в браузере, создать пулл реквест

10 ожидать код ревью

11 если нет замечаний, press merge pull request

12 выполнить 1-2 пункт

13 Убедиться, что все ок посмотрев глазами на дерево коммитов 
git log --graph --oneline --all
